# Space Shooter

Simple space shooter built with [Bevy](https://bevyengine.org/) 0.18 within a few hours.

## Play

[Play in your browser](https://cynthiaerdos.github.io/space-shooter/) or clone and `cargo run`

## Controls

| Key             | Action     |
| --------------- | ---------- |
| A / Arrow Left  | Move left  |
| D / Arrow Right | Move right |
| Space           | Shoot      |
